Regular expressions

January 24, 2015

Knowledge about Regular Expressions come in handy when working with string on advanced level. Regular Expression (regexp) is a search query, and it is formatted as a string that describes a pattern for matching elements in other strings.

Regular Expression is used with string-formatting methods, lile: 

        string = "I am a simple string"
        puts string.sub(/..$/,'blah')
        =>'I am a simple striblah'

Here /..$/ is a regex, where '.' stands for character, and '$' - for the end of the line. Similarly, /^../ means 'first 2 characters in the line'

So, as previous examples show, we use regexp to find certain patterns in strings ( to delete those patterns, replace them,print them out, store somewhere, etc.)

We can also iterate through a string, like: 

        string.scan(/./) {|letter| puts letter}

We can also scan for any amount of characters:

'string.scan(/.../)' - scans for 3 characters;

'string.scan(/\w\w/)'-scans for any two letters/digits/underscore characters in a row

for regexp, there are special characters that are denoted with '\', each of them has special meaning: 

      ^ - anchor for the beginning of a line

      $ - anchor for the end of a line

      \A - anchor for the start of a string

      \Z - anchor for the end of the string

      . - any character

      \w - any letter/digit/underscore

      \W - anything that '\w' does not match

      \d - any digit

      \D - any non-digit

      \s - whitespace (including tabs, newline, etc.)

      \S - non-whitespace

Let's say, we want to iterate through words/numbers in a string, and of course words in a string will have different length. For this purpose, we should use '+' , that follows '\w' or '\d' character. '+' matches as many characters in a row of the same type as it finds: 

      "this is a test".scan(/\w+/) { |word| p word}
      => this
         is
         a
         test

Other useful modifiers are:

{x} - matches x occurences of preceding character

{x,y} - matches at leat x, at most y characters

Character class- explicit list of all characters put inside the regexp in []. For example, to scan through all vowels: 

      string.scan(/[aeiou]/) {|x| p x}

To scan through range of characters: 

      string.scan(/[a-f]/) {|x| p x}

To check whether string matches the pattern: 

      string =~/[pattern]/

'~' is called 'matching operator' . It returns the position of the first match ( or nil, if match is not found).

To look for all but certain characters, we can negate character class. For example, to look for all characters except vowels: 

      string =~/[^aeiou]/
